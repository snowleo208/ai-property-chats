## Getting Started

### Build

To build the application, run:

```sh
pnpm build
```

### Develop

To start the development server, run:

```sh
pnpm dev
```

## Testing

This repository includes examples of testing components and functionality powered by AI. To run tests:

```sh
pnpm test
```
